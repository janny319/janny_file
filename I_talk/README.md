# Chatting Front-end 프로젝트 구성

- React - 사용언어
- Vite - 빌드관리
- Redux,Redux-toolkit - 상태광리
- recharts - 차트

## Project Structure
```
src
├── api // 공통 API 
├── assets // 컴포넌트 내부 파일
├── components // 재사용 컴포넌트
├── constants // 상수
├── hooks // 공통 Custom Hooks
├── pages // 라우팅 페이지
├── store // 공통 스토어
├── style // 스타일
└── utils // 공통 함수
```
